# image-mean
・処理  
　opencvを用いてノートパソコンのカメラから1フレーム分の画像を取得する  
　その画像の輝度の平均値計算しグラフにプロットし値、グラフを出力する  
　これをループさせる

・使い方  
　プログラムを起動したら自動でカメラが起動するので撮りたい画像をとる  
　「q」キーを押して画像の取得をやめてグラフを表示する  
 
ライブラリ　          バージョン  
matplotlib           3.0.3     
numpy                1.16.2         
opencv-python        4.1.0.25
